# Python Programming

Python is an open-source, functional programming language utilized in the development of many scientific softwares and algorithms. 

Participants in this hackathon are encouraged to have basic experience with Python. Participants should familiarize themselves with the materials included in [Python novice gapminder](https://swcarpentry.github.io/python-novice-gapminder/) and [Python novice inflammation](https://swcarpentry.github.io/python-novice-inflammation/).
