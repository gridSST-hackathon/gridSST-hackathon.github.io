## Git and GitHub

The Grid SST Hackathon will be utilizing GitHub for collaborating on code during and after the event. All participants have been invited to join the [Grid SST GitHub organization](https://github.com/gridSST-hackathon), please familiarize yourself with some of the resources below if you are rusty on Git and GitHub.


- [Introduction to GitHub](https://github.com/skills/introduction-to-github) - Get started using GitHub in less than an hour.
- [GitHub Skills](https://github.com/skills) - GitHub has a collection of beginner-friendly tutorials to help you pick up various skills.
